# Lingua - Language Learning App

A mobile-first language learning app with flashcards, spaced repetition, and matching games.

## Features

- Daily vocabulary lessons with flashcards
- Spaced repetition for optimal review scheduling
- Matching game to reinforce learning
- Progress tracking and stats
- Streak system

## Setup

1. **Create Supabase project** at supabase.com
2. **Run SQL migration** in Supabase SQL Editor
3. **Configure environment**:
   ```bash
   cp .env.example .env
   ```
4. **Edit .env with your Supabase credentials**
5. **Run**:
   ```bash
   npm install
   npm start
   ```
